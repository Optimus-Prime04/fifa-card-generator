# ⚽ FIFA Card Generator

A FIFA-style football card generator built using HTML, CSS, and JavaScript.

Users can create custom football cards by entering:

- Player Name
- Overall Rating
- Position
- Player Stats
- Custom Player Image

The generated card can also be downloaded as a PNG image.

---

# 🌍 Live Website

🔗 https://fifa-card-generator-gamma.vercel.app/

---

# 🎨 Card Types

This project currently supports:

- 🟨 Gold Card
- 🏆 Icon Card
- 🔵 UCL Card

Each card has its own custom layout and styling.
