# ⚽ FIFA Card Generator

A FIFA-style football card generator built using HTML, CSS, and JavaScript.

Users can create custom football cards with:
- Player name
- Overall rating
- Position
- Stats
- Uploaded player image
- Multiple card designs

## 🎨 Card Types

- Gold Card
- Icon Card
- UCL Card

## 🚀 Features

- Custom player cards
- Image upload system
- Download card as PNG
- Multiple FIFA-style templates
- Beginner-friendly frontend project

## 🛠️ Built With

- HTML
- CSS
- JavaScript
- html2canvas
