# FIFA Card Generator ⚽

This is a FIFA card generator I made using HTML, CSS, and JavaScript.

User can:
- upload a player image
- enter player stats
- generate your own FIFA card
- download the card as a PNG 

## Live Website

https://fifa-card-generator-gamma.vercel.app/

## Card Types

Currently i have added:
- Gold Card
- Icon Card
- UCL Card
